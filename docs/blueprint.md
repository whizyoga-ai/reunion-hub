# **App Name**: Reunion Hub

## Core Features:

- User Registration: Allow users to register with their name, phone number, and t-shirt size for the reunion.
- Reunion Details: Display the reunion details (date, time, venue: 'Alo Ghar' Uttarpara, WB, India) on the landing page.
- Icebreaker Suggestion: AI-powered tool to generate personalized icebreaker questions for attendees based on their registration details.
- FAQ Section: Implement a FAQ page based on common reunion queries.

## Style Guidelines:

- Primary color: Muted purple (#9D69A3) to convey nostalgia and sophistication.
- Background color: Light grey (#F0F0F0), a desaturated form of the primary, for a clean, modern look.
- Accent color: Soft lavender (#D6B4FC) to highlight key information and interactive elements.
- Headline font: 'Playfair', a modern sans-serif with an elegant feel; for body text: 'PT Sans', a friendly, modern sans-serif.
- Use simple, outlined icons for easy navigation. Icons should represent different sections or functionalities.
- A clean, single-page layout to present all essential information without overwhelming the user.
- Subtle transitions and animations to provide visual feedback and enhance user experience.
